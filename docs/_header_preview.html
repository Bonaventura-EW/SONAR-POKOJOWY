PO — propozycja ujednolicona

Jeden szablon nagłówka dla wszystkich zakładek. Radar SVG po lewej + nazwa marki + separator + nazwa zakładki. Po prawej pełna nawigacja, aktywna zakładka wyróżniona inset-shadow + jaśniejszym tłem. "Pominięte" dostaje dashed border żeby zaznaczyć tymczasowość.

Decyzje projektowe:
  • Logo: animowany radar SVG (z analytics/monitoring/ostatnie) — najbardziej rozpoznawalny, identyfikuje markę.
  • Tytuł: "SONAR POKOJOWY · [emoji] [Zakładka]" — marka subtelna, nazwa zakładki dominuje.
  • Nawigacja: pełna na każdej zakładce (8 linków), żeby user nie był uwięziony w sub-stronie.
  • Aktywna: jaśniejsze tło + dolna kreska, jednoznaczne wskazanie gdzie jesteś.
  • Tło: jednolity purple gradient (#667eea→#764ba2) — Firmy tracą żółty motyw na rzecz spójności.
  • "Pominięte": zachowuje dashed border jako sygnał "to jest debug/tymczasowe".
  • Responsywność: nav się wraps na wąskim ekranie, brand zostaje po lewej.
Co dotykam w implementacji:
  • analytics.html, monitoring.html, market_analysis.html, top5.html, ostatnie.html, profile_tracker.html, skipped_debug.html — wymiana sekcji <header> i powiązanych styli.
  • Strona pochodna od generatora? Sprawdzę czy któreś z nich nie są regenerowane przez Python — jeśli tak, zmiana idzie do generatora.
  • index.html — czy chcesz żeby też dostał ten sam header (na razie nie ma standardowego nagłówka, tylko mapa full-screen)?
