Widocznych ofert: 15
Średnia cena: 710 zł
Najtańsza: 610 zł
Najdroższa: 800 zł

💡 Tip: Najedź myszką na statystyki po prawej stronie!

📋 Szczegółowa lista zmian:

  • Font size wartości: 13px → 18px (+38% większe)
  • Font weight wartości: bold → 700 (wyraźniejsze)
  • Struktura: Każda statystyka jako osobna karta zamiast inline
  • Tło karty: rgba(255,255,255,0.15) (przezroczyste białe)
  • Padding karty: 12px 16px (więcej przestrzeni)
  • Border-radius karty: 8px (zaokrąglenia)
  • Ikona przed etykietą: Kropka 6px dla lepszej czytelności
  • Hover effect: transform: translateX(-2px) + jaśniejsze tło
  • Transition: all 0.3s (płynne animacje)

✨ Korzyści tej zmiany:

  • Większe liczby = łatwiejsze czytanie (szczególnie dla osób starszych)
  • Karty = wyraźne oddzielenie każdej statystyki
  • Hover effect = interaktywność i nowoczesny wygląd
  • Kropki = wizualna hierarchia i elegancja
  • Animacje = żywy, responsywny interfejs
